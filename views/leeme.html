Leeme

Este proyecto se monta usando el archivo package.json que instalará las dependencias necesarias para arrancarlo usando el comando npm install(o npm i).

Una vez completado el proceso de instalación, el comando npm run start arrancará el proyecto en el puerto 5000 de localhost y json-server en el puerto 3000.

Por último, si queremos modificar el código, debemos recompilar y convertir a JavaScript los archivos Typescript. Este es el motivo del archivo tsconfig.json. Podemos hacerlo de forma automática en Visual Studio Code pulsando en Terminal -> Ejecutar tarea... -> tsc:seguir - tsconfig.json

Contenidos

  • Filter-Map-Reduce: Un ejercicio para practicar programación funcional. Utiliza un objeto JSON con contratos para listar y extraer de él algunos datos usando funciones filter, map, reduce, find, Set, etc.
  • Libros: Un CRUD(Create, Read, Update, Delete) de libros usando json-server
  • Ejercicios ES6: Un repaso a las características introducidas con ECMAScript 2015
  • Películas Un CRUD de películas esta vez usando clases de Typescript a modo de POJOs (clase Pelicula y clase FormatosPelis)
