Código inicial:

Getters y Setters para el control de acceso a un objeto

Código inicial: 


            /* Alter code below this line */

            /* Alter code above this line */
            
            const thermos = new Thermostat(76); // setting in Fahrenheit scale
            let temp = thermos.temperature; // 24.44 in C
            thermos.temperature = 26;
            temp = thermos.temperature; // 26 in C

Solución: 


            /* Alter code below this line */
            class Thermostat{
                constructor(temperature){
                    this._temperature = temperature;
                }
            
                get temperature(){
                    this._temperature = 5/9 * (this._temperature - 32);
                    return  this._temperature;
                }
            
                set temperature(temperature){
                    this._temperature = temperature;
                }
            }
            /* Alter code above this line */
            
            const thermos = new Thermostat(76); // setting in Fahrenheit scale
            let temp = thermos.temperature; // 24.44 in C
            thermos.temperature = 26;
            temp = thermos.temperature; // 26 in C

Crear una Promesa

Una promesa es un objeto Promise que se instancia pasándole dos parámetros obligatorios: resolve y reject


            const makeServerRequest = new Promise((resolve, reject) => {

            });

Completar la promesa con resolve y reject

Código inicial: 


                const makeServerRequest = new Promise((resolve, reject) => {
                    // responseFromServer represents a response from a server
                    let responseFromServer;
                      
                    if(responseFromServer) {
                      // change this line
                    } else {  
                      // change this line
                    }
                  });

Solución: 


                const makeServerRequest = new Promise((resolve, reject) => {
                    // responseFromServer represents a response from a server
                    let responseFromServer;
                      
                    if(responseFromServer) {
                      resolve("We got the data");
                    } else {

                  });

Problema matemático (Map, Filter, Reduce)

Tenemos una funcion squareList a la que se le pasa un array de numeros reales. Queremos que devuelva un array con los cuadrados de cada número positivo entero

Solución: 


        const squareList = (arr) => {
          // only change code below this line
        
          return arr.filter(num => num > 0 && Number.isInteger(num) ).map(num => Math.pow(num, 2));
        
          
          // only change code above this line
        };
        
        // test your code
        const squaredIntegers = squareList([-3, 4.8, 5, 3, -3.2]);
        console.log(squaredIntegers);
